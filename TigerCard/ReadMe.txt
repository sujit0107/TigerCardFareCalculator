Instructions: 

1. The program is written in .net Core 3.1 so its platform independent. 
2. Input can be provided in InputFile.txt, few scenarios are already included in the input file. More scenarios could be appended in the same file.
3  Each scenarion should be seperated by a row of "------". The program will calculate the total fare in each scenario and print it.
4. Follow the template in input file i.e Columns provided in input should be have the sequence Date, Time, FromZone, ToZone.And each column should be seprated by "|"   
3. Input can also be provided in form of test cases in the test project. In test project there is a class "ScenarioTests.cs" where more scenario can be added.
4. All the configurations which is required to calculate the total fare is maintained in Rules.xml.
